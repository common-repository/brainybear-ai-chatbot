== Installation ==

Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.

After active plugin go to Brainybear->Settings to setup your API key. 

== Third party services ==

This plugin will connect to Brainybear backend API in order to identify user ID as well as fetching the correct assistant.

The API being called is: https://api.brainybear.ai/user/wp-api?key=YOUR_API_KEY

Read more about the [Terms of Use](https://brainybear.ai/terms-of-service) and [Privacy Policy](https://brainybear.ai/privacy) about third party links.
